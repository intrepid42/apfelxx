The evolsetup class contains all the parameters needed to perform the DGLAP evolution. The constructor initializes a set of default parameters which correspond to the LH benchmark parameters at NNLO.

Include only once

#pragma once

#include <string>

Solution of the DGLAP equation

  };

  };
